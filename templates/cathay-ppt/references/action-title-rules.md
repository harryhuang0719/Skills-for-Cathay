# Action Title 规则

> 来源：aham-ppt brand spec + IB/PE IC memo 惯例。
>
> **核心**：每页标题是**完整结论句**，不是主题词。读者扫 titles 就能读懂整个 deck。

---

## 为什么

banker 读 deck 的方式：90% 人只扫标题和表头，10% 人读 bullet。如果标题只是"市场分析"这种主题词，扫的人什么都没看到，deck 就废了。

测试法：**只看每一页的标题，能不能复述整份 deck 的主要论点？** 可以 = 合格，不可以 = 标题全部要改。

---

## 规则

### R1：标题必须是完整的句子，带明确结论

| ❌ 主题词 | ✅ 结论句 |
|-----------|----------|
| 市场分析 | 海洋机器人市场 2028 年 2000 亿，国产替代空间 40% |
| 财务表现 | 2023-2025 收入 CAGR 180%，毛利从 18% 提升至 42% |
| 投资亮点 | 3 个支柱：技术壁垒 / 客户锁定 / 商业化拐点 |
| 公司介绍 | Sealien：2021 成立、80 人团队、研发占 60% |
| 风险 | 核心风险 3 项均已对冲 |
| 退出路径 | Base 2030 IPO 25x PE，对应 IRR 28% / MOIC 4.2x |

### R2：长度上限

- 中文：**≤ 28 字**（超长 banker 扫一眼读不完，且不好排版）
- 英文：**≤ 14 words**
- 标题一行放不下就是还没提炼干净，不是字号问题

### R3：标题必须自带数字

如果内容里有数字（增速、金额、估值、TAM），**至少 1 个要出现在标题**。banker 记得住的只有数字。

```
❌ Sealien 业务增长迅速
✅ Sealien 2023-2025 收入 CAGR 180%
```

### R4：视觉 anchor（可选，推荐）

在标题左侧加 **3pt Maroon (#800000) 竖线**作为 anchor 元素。不是装饰，是告诉读者"这是结论不是内容"。

`lib/text_engine.py` / `slide_templates.py` 里的 T1-T16 默认会加这条线；只有自己手搓 slide 时需要注意。

### R5：字号和字重

- 中文标题：**18-22pt 粗体 Maroon** (#800000)
- 英文标题：**18-20pt Bold Calibri Maroon**
- 封面标题例外：32-40pt
- **不要**用第二种颜色作为标题强调色（禁止 titre rouge + titre doré 混用）

### R6：禁止的标题形式

| 禁止 | 原因 |
|------|------|
| "XX 概况"、"XX 分析"、"关于 XX" | 主题词，无结论 |
| "多维度深入剖析 XX" | 形容词堆砌，没有信息 |
| "赋能 / 颠覆 / 一站式 / 全链路 XX" | 禁用词 |
| 结尾带"？"的疑问句 | banker deck 不是辩论稿 |
| 纯英文大写标题 | banker 读得累 |

### R7：例外页面

以下页面可以不遵循 R1（完整结论句），但仍要遵循 R5（字号字重）：

- **封面**：用公司名 + 副标题（e.g., "Project Sealien · 海洋机器人投资推介"）
- **目录**：标题就是"目录 / Contents / Agenda"
- **章节 divider**：用章节编号 + 章节名（e.g., "01 · 市场与赛道"）
- **Q&A / Thank You 末页**

## Ghost Deck 阶段的标题写法

在方法论阶段 4（Ghost Deck）写每页标题时：

1. **先写一句话想表达什么**（口头表述，自由形式）
2. **压缩成 ≤ 28 字的句子**（必要时砍形容词）
3. **检查有没有数字**（如果内容里有，挪进标题）
4. **检查有没有结论**（不是"分析 X"而是"X 是 Y"）

Ghost Deck 全部页标题写完一轮后，把所有标题抄出来当一段话读，如果这段话能基本复述 deck 论点就合格。

## 反例复盘

**反例 1**（原版 / 改后）：

- ❌ "全球海洋机器人市场空间测算"  
- ✅ "全球海洋机器人 TAM 800 亿美元，CAGR 25%，中国占 40%"

改动点：加了 3 个关键数字、去掉"测算"这个过程词。

**反例 2**：

- ❌ "关于 Sealien 团队的多维度介绍"  
- ✅ "Sealien 团队：CTO 中科院背景 + 3 位核心骨干均来自头部企业"

改动点：具体化、去掉"多维度"这种形容词。

## Pre-ship checklist

- [ ] 所有非例外页标题是完整句子（有主语 + 谓语 + 结论）
- [ ] 所有含数字的内容页，标题里出现至少 1 个数字
- [ ] 所有标题 ≤ 28 字
- [ ] 所有标题颜色都是 Maroon，不是黑色也不是 Gold
- [ ] 所有标题字号 18-22pt（封面除外）
- [ ] 把所有标题抄出来读一遍，能复述 deck 论点
